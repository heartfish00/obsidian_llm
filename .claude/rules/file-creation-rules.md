# File Creation Rules

## Note Location

새 노트는 볼트 루트에 직접 생성:
- **경로**: `D:\Sales Planning\obsidian\my\`
- **파일명**: `YYYY-MM-DD-description.md`
- 연도가 지나면 `PARA_3Resources\연도\` 로 이동 (사용자가 수동 처리)

## Code Output Location

코드·스크립트 산출물은 `00. Inbox/` 하위에 저장.

## File Naming Convention

- 날짜 포함: `YYYY-MM-DD-description.ext`
- 설명적인 이름 사용

## Multi-File Project Folder Rule

다중 파일 프로젝트는 중간 폴더를 먼저 생성:

```
00. Inbox/
└── 2026-05-13-project-name/
    ├── index.html
    ├── styles.css
    └── script.js
```

관련 파일을 폴더 루트에 직접 흩뿌리지 말 것.
